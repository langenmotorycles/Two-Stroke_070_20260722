# LANGEN_TWO-STROKE_070_CERTIFIED_PRE-OWNED_Langen_Flipbook

This folder is ready to publish with GitHub Pages.

## Publish on GitHub

1. Create a new GitHub repository.
2. Extract this ZIP.
3. Upload all extracted files to the root of the repository.
4. Commit the files.
5. Open **Settings → Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select the **main** branch and **/(root)** folder, then save.
8. GitHub will display the public website address after deployment.

The flipbook is contained in `index.html`. The original PDF is embedded in that file, so no separate PDF upload is required.
